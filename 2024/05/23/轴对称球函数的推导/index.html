<轴对称球函数的推导>

轴对称球函数的推导

引言

在球坐标系中，拉普拉斯方程 (\nabla^2 u = 0) 是描述许多物理现象（如电场和引力场）的基本方程。我们可以通过分离变量法将该方程拆解为径向部分和角度部分，进而求解。本文详细推导了在轴对称情况下（即 (m = 0)）的解。

拉普拉斯方程在球坐标系中的形式

在球坐标系 ((r, \theta, \phi)) 中，拉普拉斯方程的表达式为：
[ \nabla^2 u = \frac{1}{r^2} \frac{\partial}{\partial r} \left( r^2 \frac{\partial u}{\partial r} \right) + \frac{1}{r^2 \sin \theta} \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial u}{\partial \theta} \right) + \frac{1}{r^2 \sin^2 \theta} \frac{\partial^2 u}{\partial \phi^2} ]

分离变量法

假设解 ( u(r, \theta, \phi) ) 可以表示为径向部分和角度部分的乘积：
[ u(r, \theta, \phi) = R(r) Y(\theta, \phi) ]

将该形式代入拉普拉斯方程，展开并分离变量。

径向部分的展开

[ \frac{\partial (R(r) Y(\theta, \phi))}{\partial r} = \frac{dR(r)}{dr} Y(\theta, \phi) ]
[ \frac{\partial^2 (R(r) Y(\theta, \phi))}{\partial r^2} = \frac{d^2R(r)}{dr^2} Y(\theta, \phi) ]

角度部分的展开

[ \frac{\partial (R(r) Y(\theta, \phi))}{\partial \theta} = R(r) \frac{\partial Y(\theta, \phi)}{\partial \theta} ]
[ \frac{\partial^2 (R(r) Y(\theta, \phi))}{\partial \theta^2} = R(r) \frac{\partial^2 Y(\theta, \phi)}{\partial \theta^2} ]
[ \frac{\partial^2 (R(r) Y(\theta, \phi))}{\partial \phi^2} = R(r) \frac{\partial^2 Y(\theta, \phi)}{\partial \phi^2} ]

代入拉普拉斯方程后得到：
[ \nabla^2 u = \frac{1}{r^2} \left[ Y \frac{d}{dr} \left( r^2 \frac{dR}{dr} \right) \right] + \frac{R}{r^2 \sin \theta} \left[ \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial Y}{\partial \theta} \right) \right] + \frac{R}{r^2 \sin^2 \theta} \left[ \frac{\partial^2 Y}{\partial \phi^2} \right] ]

分离变量

因为左边第一项仅依赖 ( r )，而右边第二项仅依赖 ( \theta ) 和 ( \phi )，这意味着两边都必须等于一个常数。我们称之为分离常数 (-l(l+1))：
[ \frac{1}{R} \frac{d}{dr} \left( r^2 \frac{dR}{dr} \right) = l(l+1) ]
[ \frac{1}{Y} \left[ \frac{1}{\sin \theta} \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial Y}{\partial \theta} \right) + \frac{1}{\sin^2 \theta} \frac{\partial^2 Y}{\partial \phi^2} \right] = -l(l+1) ]

径向方程

径向方程为：
[ \frac{d}{dr} \left( r^2 \frac{dR}{dr} \right) = l(l+1) R ]

将其展开得到：
[ r^2 \frac{d^2R}{dr^2} + 2r \frac{dR}{dr} - l(l+1)R = 0 ]

这是一个常系数的二阶齐次线性微分方程。我们可以假设解的形式为：
[ R(r) = r^n ]

将假设的解代入径向方程：
[ r^2 \frac{d^2}{dr^2} (r^n) + 2r \frac{d}{dr} (r^n) - l(l+1) r^n = 0 ]

计算导数：
[ \frac{d}{dr} (r^n) = n r^{n-1} ]
[ \frac{d^2}{dr^2} (r^n) = n(n-1) r^{n-2} ]

代入径向方程：
[ r^2 n(n-1) r^{n-2} + 2r n r^{n-1} - l(l+1) r^n = 0 ]
[ n(n-1) r^n + 2n r^n - l(l+1) r^n = 0 ]
[ [n(n-1) + 2n - l(l+1)] r^n = 0 ]

因为 ( r^n ) 非零，所以系数部分必须为零：
[ n(n-1) + 2n - l(l+1) = 0 ]
[ n^2 + n - l(l+1) = 0 ]

这是一个关于 ( n ) 的二次方程，解得：
[ n = \frac{-(1) \pm \sqrt{(1)^2 - 4 \cdot 1 \cdot (-l(l+1))}}{2 \cdot 1} ]
[ n = \frac{-1 \pm \sqrt{1 + 4l(l+1)}}{2} ]
[ n = \frac{-1 \pm \sqrt{1 + 4l^2 + 4l}}{2} ]
[ n = \frac{-1 \pm \sqrt{(2l+1)^2}}{2} ]
[ n = \frac{-1 \pm (2l+1)}{2} ]

所以，两个解分别为：
[ n = l ]
[ n = -(l+1) ]

因此，径向方程的通解为：
[ R(r) = A r^l + B r^{-(l+1)} ]
其中，( A ) 和 ( B ) 是待定常数。

角度方程

角度方程为：
[ \frac{1}{\sin \theta} \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial Y}{\partial \theta} \right) + \frac{1}{\sin^2 \theta} \frac{\partial^2 Y}{\partial \phi^2} + l(l+1)Y = 0 ]

假设角度部分的解可以表示为：
[ Y(\theta, \phi) = \Theta(\theta) \Phi(\phi) ]

代入角度方程：
[ \frac{1}{\sin \theta} \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial (\Theta(\theta) \Phi(\phi))}{\partial \theta} \right) + \frac{1}{\sin^2 \theta} \frac{\partial^2 (\Theta(\theta) \Phi(\phi))}{\partial \phi^2} + l(l+1) \Theta(\theta) \Phi(\phi) = 0 ]

将 ( \Theta ) 和 ( \Phi ) 分开：
[ \frac{\Phi(\phi)}{\sin \theta} \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial \Theta}{\partial \theta} \right) + \frac{\Theta(\theta)}{\sin^2 \theta} \frac{\partial^2 \Phi}{\partial \phi^2} + l(l+1) \Theta(\theta) \Phi(\phi) = 0 ]

两边同时除以 ( \Theta(\theta) \Phi(\phi) \sin^2 \theta )：
[ \frac{1}{\Theta} \frac{1}{\sin \theta} \frac{d}{d \theta} \left( \sin \theta \frac{d \Theta}{d \theta} \right) + \frac{1}{\Phi} \frac{d^2 \Phi}{d \phi^2} + l(l+1) = 0 ]

这就得到了分离的角度方程：

(\Phi) 部分

[ \frac{d^2 \Phi}{d \phi^2} = -m^2 \Phi ]

这是一个简单的二阶常系数微分方程，解为：
[ \Phi(\phi) = C e^{im\phi} + D e^{-im\phi} ]

为了满足周期性条件 ( \Phi(\phi + 2\pi) = \Phi(\phi) )，我们要求 ( m ) 为整数。

(\Theta) 部分

当 ( m = 0 ) 时，角度方程简化为：
[ \frac{1}{\sin \theta} \frac{\partial}{\partial \theta} \left( \sin \theta \frac{\partial \Theta}{\partial \theta} \right) + l(l+1) \Theta = 0 ]

令 ( x = \cos \theta )，则有 ( \sin \theta = \sqrt{1 - x^2} )，并且 ( \frac{d}{d\theta} = -\sqrt{1 - x^2} \frac{d}{dx} )。方程变为：
[ \frac{d}{dx} \left( (1 - x^2) \frac{d \Theta}{dx} \right) + l(l+1) \Theta = 0 ]

这是勒让德方程的标准形式：
[ (1 - x^2) \frac{d^2 P_l}{dx^2} - 2x \frac{d P_l}{dx} + l(l+1) P_l = 0 ]

其解为勒让德多项式 ( P_l(x) )，解的形式为：
[ P_l(x) = \sum_{k=0}^{\left\lfloor \frac{l}{2} \right\rfloor} \frac{(-1)^k (2l - 2k)!}{2^l k! (l - k)! (l - 2k)!} x^{l - 2k} ]

轴对称球函数

在轴对称情况下（( m = 0 )），球谐函数化简为勒让德多项式乘以一个常数：
[ Y(\theta, \phi) = \text{常数} \cdot P_l(\cos \theta) ]

这个 ( Y(\theta, \phi) ) 仅依赖于 ( \theta )，与 ( \phi ) 无关，因此它是轴对称的。

总解

结合径向部分和角度部分的解，拉普拉斯方程的总解为：
[ u(r, \theta, \phi) = \left( A r^l + B r^{-(l+1)} \right) Y_l^m(\theta, \phi) ]
[ = \left( A r^l + B r^{-(l+1)} \right) P_l^m(\cos \theta) e^{im\phi} ]

在轴对称情况下（( m = 0 )），解为：
[ u(r, \theta, \phi) = \left( A r^l + B r^{-(l+1)} \right) P_l(\cos \theta) ]

通过以上推导，我们得到了在球坐标系中拉普拉斯方程的轴对称解。这些解在电场、引力场等物理问题中有广泛应用。

总结

本文详细推导了在球坐标系中拉普拉斯方程 (\nabla^2 u = 0) 的分离变量解法，并给出了在轴对称情况下的具体解。通过分离变量法，我们将方程拆解为径向部分和角度部分，并分别求解，最终得到径向和角度部分的解析解。
